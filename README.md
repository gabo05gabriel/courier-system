# Courier System

Sistema web y móvil para Courier Bolivian Express S.R.L.

## Tecnologías utilizadas

- Node.js + Express (backend)
- PostgreSQL (base de datos)
- Flutter (aplicación móvil y web)
- Docker (despliegue)
- GitHub (control de versiones)

## Funcionalidades principales

- Seguimiento de paquetes en tiempo real
- Rutas inteligentes para mensajeros
- Reportes operativos
- Panel administrativo y cliente
